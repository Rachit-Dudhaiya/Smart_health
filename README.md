# Smart Health

Smart Health is a healthcare dashboard concept for PHC/CHC facilities. It provides a frontend structure for real-time monitoring of:

- Medicine stock
- Patient flow
- Bed availability
- Doctor attendance

## Project Structure

- index.html — Home page
- pages/medicine-stock.html — Medicine stock page
- pages/patient-flow.html — Patient flow page
- pages/bed-availability.html — Bed availability page
- pages/doctor-attendance.html — Doctor attendance page
- pages/contact.html — Contact page
- styles.css — Shared styling
- script.js — Small interactive behavior

## How to View

Open the project folder in a browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```
